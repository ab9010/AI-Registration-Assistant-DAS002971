# AI Registration Assistant - DAS002971

I completed **AI Registration Assistant (Task AI-SS-001)** as part of the Free Online AI & Data Science Internship.

The project is a conversational chatbot that guides students through internship registration. It uses Python, NLTK, scikit-learn, Flask, HTML, CSS, JavaScript, and JSON. The system performs NLP preprocessing, machine-learning intent classification, entity extraction, input validation, dialog management, registration confirmation, FAQ handling, sentiment logging, and analytics.

## Features

- Greeting and student support
- TF-IDF and Logistic Regression intent recognition
- Name and email entity extraction
- Multi-step registration flow
- Name, email, program, year, and experience validation
- Unique registration confirmation ID
- FAQ handling
- English and basic Hindi/Hinglish support
- Sentiment analysis and chat logging
- Responsive Flask web interface
- Admin analytics dashboard
- JSON storage without a database
- Automated tests
